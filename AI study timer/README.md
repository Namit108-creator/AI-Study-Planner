# AI Study Planner

An AI-powered study planner built using Python, Flask, Machine Learning, SQLite, HTML, CSS, and JavaScript.

## Features

- AI-based study recommendations
- Weekly and monthly study scheduling
- Login and registration system
- Productivity score tracking
- Pomodoro timer
- Voice assistant integration
- Subject weakness analysis
- Charts and progress analytics
- SQLite database support

## Technologies Used

- Python
- Flask
- SQLite
- scikit-learn
- HTML/CSS/JavaScript
- Pandas
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run the App

```bash
python app.py
```

Then open:

```text
http://127.0.0.1:5000
```
